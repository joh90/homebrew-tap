# Homebrew Tap

Homebrew formulae for [joh90](https://github.com/joh90) projects.

## Installation

```bash
brew tap joh90/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| `slackasme` | Slack CLI - interact with Slack as yourself |

### slackasme

```bash
brew install slackasme
```

See [joh90/slackasme](https://github.com/joh90/slackasme) for usage.
