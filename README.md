# Homebrew Tap

Homebrew formulae for [joh90](https://github.com/joh90) projects.

## Installation

```bash
brew tap joh90/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| `slack-cli` | Slack CLI - interact with Slack as yourself |

### slack-cli

```bash
brew install slack-cli
```

See [joh90/slack-cli](https://github.com/joh90/slack-cli) for usage.
